About

Movie Logger is a website made by Shubhan Bhasin during a bootcamp from TS. It is made fully using HTML, CSS and JS. Movie Logger has a a goal of saving all the movies in one place (in this case local storage of your browser) so that you never forget which movie you want to watch when you get home. Or the movies you have allready watched and you want to share them with friends and family. Please dont copy this website or idea. Try making your own with VS Code (I used it too).
